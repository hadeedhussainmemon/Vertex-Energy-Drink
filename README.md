## ‚ö° Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/your-username/vertex-energy.git
cd vertex-energy
npm install
```

### 2. Environment Setup
Create a `.env.local` file based on `.env.example`:
```bash
cp .env.example .env.local
```
Fill in your `MONGO_URI` and `JWT_SECRET`.

### 3. Run Locally
```bash
npm run dev
```
Visit `http://localhost:3000` to witness the future.
